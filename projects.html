Rock-Paper-Scissor (2025)

Stack: DOM,EventListner,HTML,CSS,Javascript

Developed a fully interactive Rock-Paper-Scissors game using HTML, CSS, and JavaScript that allows users to play against the computer with a dynamic and engaging interface. The game features a “Rock… Paper… Scissors…” countdown animation, real-time score tracking, and a 3D flip reveal effect to display both the player’s and computer’s choices. It also includes an initial user confirmation popup, a dark-themed UI with glassmorphism effects, and smooth CSS transitions for an enhanced gaming experience.

Quiz Website(2025)

Stack: JavaScript, HTML, CSS

Built an interactive Quiz Application using HTML, CSS, and JavaScript that dynamically presents multiple-choice questions with a built-in timer, score tracker, and animated transitions. The app begins by asking users if they want to play, then runs through a set of 10 JavaScript, HTML, and CSS-based questions, providing instant feedback for each answer. It includes smooth animations, modern UI design, and a final score screen for a complete user experience.
